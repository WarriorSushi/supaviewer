# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest `main` | Yes |
| Older commits | No |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public issue.
2. Email **supaviewer@proton.me** with details of the vulnerability.
3. Include steps to reproduce if possible.

### What to expect

- **Acknowledgment** within 48 hours.
- **Assessment** within 7 days.
- **Fix or mitigation** as soon as reasonably possible, depending on severity.

We take security seriously and appreciate your help keeping Supaviewer and its users safe.
